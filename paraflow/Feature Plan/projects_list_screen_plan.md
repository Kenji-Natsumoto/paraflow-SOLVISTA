# プロジェクト一覧ページ
ユーザーが管理しているすべての社会課題プロジェクトを一覧表示し、新規プロジェクト作成やプロジェクト詳細へのアクセスを可能にする。

Layout Hierarchy:
- Header (Full-width):
  - トップナビゲーションバー
- Content Container (Positioned below the header):
  - メインコンテンツエリア

## トップナビゲーションバー
- ロゴ表示
- グローバル検索バー
- 通知アイコン
- 設定アイコン
- ユーザーアバター

## メインコンテンツエリア
- ページタイトル:
  - 「プロジェクト一覧」表示
  - 新規プロジェクト作成ボタン
- フィルター・ソートバー:
  - プロジェクト検索機能
  - ステータスフィルター（全て/進行中/完了/アーカイブ）
  - ソート機能（更新日順/作成日順/名前順）
  - 表示切り替え（カードビュー/リストビュー）
- プロジェクトカード表示エリア:
  - プロジェクトカード複数表示（各カードに以下を含む）:
    - プロジェクト名
    - プロジェクト説明（最初の100文字）
    - ステータスラベル
    - 最終更新日時
    - 課題数（ステークホルダー課題数/社会システム課題数）
    - コラボレーター数とアバター
    - クイックアクションアイコン（編集/共有/アーカイブ）
  - 空状態メッセージ（プロジェクトがない場合）:
    - 「まだプロジェクトがありません」メッセージ
    - 新規プロジェクト作成を促すCTA
