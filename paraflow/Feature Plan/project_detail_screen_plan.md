# プロジェクト詳細ページ
特定のプロジェクトの全体像を表示し、プロジェクト情報の編集、課題の管理、バージョン履歴の参照、チームコラボレーション、AI分析、エクスポートなどの主要機能へのアクセスを提供する。

Layout Hierarchy:
- Header (Full-width):
  - トップナビゲーションバー
- Content Container (Positioned below the header):
  - メインコンテンツエリア

## トップナビゲーションバー
- ロゴ表示
- グローバル検索バー
- 通知アイコン
- 設定アイコン
- ユーザーアバター

## メインコンテンツエリア
- プロジェクトヘッダー:
  - プロジェクト名表示
  - ステータスバッジ
  - 最終更新日時
  - プロジェクト編集ボタン
  - 共同編集招待ボタン
  - エクスポートボタン（JSONダウンロード）
- プロジェクト基本情報セクション:
  - プロジェクト説明全文
  - 作成日
  - コラボレーターリスト（アバターと名前）
  - プロジェクトタグ
- タブナビゲーション:
  - 課題タブ（選択中）
  - バージョン履歴タブ
  - 共同編集タブ
  - AI分析タブ
- 課題管理エリア（課題タブ選択時）:
  - 課題追加ボタン
  - 課題タイプフィルター（全て/ステークホルダー課題/社会システム課題）
  - 課題カテゴリーフィルター（全て/クライシス/リスク/問題/課題）
  - 課題カード複数表示:
    - 課題タイトル
    - 課題タイプラベル
    - 課題カテゴリーラベル
    - ゴール（要約）
    - 現状（要約）
    - ギャップ（要約）
    - 最終更新日時
    - クイックアクション（編集/AI分析/削除）
  - 空状態メッセージ（課題がない場合）:
    - 「まだ課題が定義されていません」メッセージ
    - 課題追加を促すCTA
